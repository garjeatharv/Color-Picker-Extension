# Color-Picker-Extension
Thank you for choosing the Color Picker Extension! This is a simple yet useful tool built using HTML and CSS that allows users to obtain the hex code for any color present on a website.

## Getting Started:
1. To install the extension, simply download the files and extract them to a folder on your computer.
2. Open your browser and navigate to the Extensions page.
3. Click on the "Load unpacked" button and select the folder containing the extracted files.
4. The Color Picker Extension will now be installed and ready to use.

## Using the Extension:
1. Navigate to any website and click on the Color Picker icon located in the top right corner of your browser window.
2. Hover over any color on the website and click on it to select the color.
3. The hex code for the selected color will now be displayed in a pop-up window.
4. Click on the "Copy" button to copy the hex code to your clipboard.
